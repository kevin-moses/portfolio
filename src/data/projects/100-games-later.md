# 100 Games Later

Design journal documenting the process of building one micro-game per
day. Each entry includes gifs, rules, player feedback, and a downloadable
prototype.

## Highlights

- Automated capture pipeline exports playable recordings nightly
- Backlog board visualizes mechanics explored vs. wishlist ideas
- Newsletter integration sends weekly retrospectives

## Tech Stack

- Astro + MDX content engine
- Godot + Playdate SDK for game builds
- Supabase for telemetry and mailing list automations

## Impact

Spawned three festival showcases and helped land consulting work with a
toy studio looking to prototype tactile storytelling experiences.
